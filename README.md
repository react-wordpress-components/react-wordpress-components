# WReact

React based theme developmental tools.

## demos

https://wreact.github.io/wreact

## development

```shell
$ git clone git@github.com:wreact/wreact.git
$ cd wreact
$ npm install
$ npm test
```

## run demos

```shell
$ git clone git@github.com:wreact/wreact.git
$ cd wreact
$ npm install
$ npm start
```

then open http://localhost:4000 with your browser.
